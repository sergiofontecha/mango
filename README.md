# MANGO
Mango Tech Test
## Prerequisites

- node v18 or higher
- npm v10.1.0 or higher
- web browser with ES5 or higher
## Node version
```
18^
```
## How run the project?

1. install dependencies

```shell
npm i
```

2. run project (Compiles and hot-reloads for development)

```shell
npm run dev
```

3.  open your favorite web browser http://localhost:3000/
